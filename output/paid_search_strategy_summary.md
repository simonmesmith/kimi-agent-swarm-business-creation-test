## OPTIMIZATION ROADMAP

### Phase 1 (Weeks 1-4): Launch & Learn
- Start with $10K/month budget
- Focus on top 3 ad groups (80% of budget)
- Monitor Quality Scores, aim for 7+
- Add search terms as negative keywords

### Phase 2 (Weeks 5-8): Optimize
- Pause keywords with CAC >$100
- Increase bids on CAC <$60 keywords
- Expand winning ad copy variations
- Launch responsive search ads

### Phase 3 (Weeks 9-12): Scale
- Increase budget to $15K if CAC <$75
- Activate Brand/Competitor campaign
- Test Performance Max campaigns
- Implement remarketing
